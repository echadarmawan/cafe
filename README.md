# Aplikasi Pemesanan Cafe
Sebuah aplikasi yang dapat digunakan untuk pemesanan minuman dan makanan pada cafe, restoran, rumah makan atau yang sejenisnya.

## Pemilik/admin
    - Melihat laporan (Report)
    - Menambah/mengubah/menghapus daftar menu dan harga (Menu List)
    - Menambah/mengubah/menghapus user (User)
    Order

## Kasir
    Order
    - Melihat harga dan total harga pesanan
    - Konfirmasi pembayaran
    - Membuat bukti pembayaran

## Pelayan
    Menu List, Order
    - Melihat menu dan harga
    - Memilih menu
    - Melihat total harga pesanan
    - Membuat pesanan
    - Update pesanan
    - Hapus pesanan

## Dapur
    Order
    - Menerima pesanan
    - Konfirmasi terima pesanan
    - Mengubah status pesanan "siap saji"
